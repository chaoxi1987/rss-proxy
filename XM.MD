# WordPress 网页计算器项目计划

## 项目概述
创建一个类似腾讯浏览器工具集的计算器工具页面，包含多种不同类型的计算器供用户选择使用。所有计算器将保持与 WebStack 主题风格一致，并且可以直接通过 WordPress 后台页面编辑器添加。

## 开发路线图

### 第一阶段：基础计算器
1. 标准计算器
   - 基本运算（加减乘除）
   - 百分比计算
   - 清除/回退功能
   - 键盘支持

2. 科学计算器
   - 包含标准计算器全部功能
   - 三角函数
   - 指数和对数
   - 平方/开方

### 第二阶段：专业计算器
1. 单位换算计算器
   - 长度换算
   - 重量换算
   - 面积换算
   - 体积换算
   - 温度换算

2. 金融计算器
   - 利率计算
   - 贷款计算（等额本息/等额本金）
   - 投资回报率计算

### 第三阶段：健康计算器
1. 健康计算器
   - BMI计算
   - 体脂率估算
   - 每日热量需求计算
   - 饮水建议

### 第四阶段：新增计算器计划
1. 货币汇率计算器
   - 实时汇率查询
   - 货币转换
   - 主流货币汇率展示

## 实现方案

### 技术架构
1. 前端实现
   - 纯 HTML、CSS 和 JavaScript
   - 响应式设计，适配各种设备
   - 模块化代码结构
   - 使用 CSS 变量匹配主题颜色

2. WordPress 集成
   - 使用 WordPress 页面编辑器的 HTML 模式
   - 每个计算器独立一个页面
   - 工具集主页使用 Grid/Flex 布局展示所有计算器入口

### 页面结构
1. 工具集主页
   ```html

     <div class="calculator-card">
       <!-- 每个计算器的入口卡片 -->
     </div>
   </div>
   ```

2. 单个计算器页面
   ```html
   <div class="calculator-container">
     <div class="calculator-header">
       <!-- 标题和说明 -->
     </div>
     <div class="calculator-body">
       <!-- 计算器主体 -->
     </div>
     <div class="calculator-footer">
       <!-- 额外信息或说明 -->
     </div>
   </div>
   ```

## 标准计算器实现说明

### WordPress页面创建步骤

1. 进入WordPress后台
2. 点击"页面" -> "新建页面"
3. 设置页面标题为"标准计算器"
4. 切换到"文本"或"代码"编辑模式
5. 粘贴`standard-calculator.html`文件内容
6. 点击"发布"或"更新"按钮保存页面

### 功能说明

1. 基本功能：
   - 加减乘除基本运算
   - 百分比计算
   - 清除(C)和回退(⌫)功能
   - 支持小数点输入
   - 支持键盘操作

2. 键盘支持：
   - 数字键(0-9)：输入数字
   - 运算符：+、-、*、/、%
   - Enter或=：计算结果
   - Backspace：删除最后一位
   - Esc：清除所有输入

3. 界面特点：
   - 响应式设计，适配各种设备
   - 清晰的显示界面，包括历史记录
   - 符合WebStack主题风格
   - 操作按钮有反馈效果
   - 包含使用说明

## 工具集主页实现说明

### WordPress页面创建步骤

1. 进入WordPress后台
2. 点击"页面" -> "新建页面"
3. 设置页面标题为"计算器工具集"
4. 切换到"文本"或"代码"编辑模式
5. 粘贴`calculator-index.html`文件内容
6. 点击"发布"或"更新"按钮保存页面

### 功能说明

1. 页面结构：
   - 响应式网格布局，自动适应不同屏幕尺寸
   - 每个计算器工具以卡片形式展示
   - 卡片包含图标、标题、描述和操作按钮

2. 卡片设计：
   - 悬停效果：卡片上移并增加阴影
   - 清晰的视觉层次：标题区、内容区和操作区
   - 使用Font Awesome图标（WebStack主题已包含）

3. 链接设置：
   - 已实现的计算器：链接到对应的计算器页面
   - 未实现的计算器：显示"即将推出"，不可点击

4. 注意事项：
   - 链接地址需要根据实际页面URL进行调整
   - 如果WebStack主题没有包含Font Awesome，可能需要额外添加

### 使用说明

1. 创建工具集主页后，需要确保标准计算器页面的URL与工具集主页中的链接一致
2. 默认链接为"/standard-calculator"，如果您的页面URL不同，请修改工具集主页中的链接
3. 随着新计算器的添加，将"即将推出"按钮替换为实际链接
